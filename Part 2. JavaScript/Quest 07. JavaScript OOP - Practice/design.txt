
Desktop Design
1. 아이콘
  - 드래그 하면서 움직인다. (o)
  - 드래그 할때 마우스가 일시적으로 아이콘을 빠져나가도 잘 움직이게끔

2. 폴더
  - 폴더인 아이콘은 더블클릭하면 폴더 창이 열린다. (o)
  - 폴더 창의 초기값은 200x200 px (o)
  - 폴더 창 끌 수 있어야 됨(더블클릭) (o)
  - 폴더 창 움직일 수 잇어야 됨. (o)
  - 폴더 창 크기 및 위치는 보존되어야 함 (o)
  - 폴더 창 크기 변경
  - 아이콘을 폴더 안에 넣을 수 있게 끔
  - 폴더와 윈도우의 경계에 있을 때 어느쪽에 위치할지 강제로 밀어야됨(두 공간에 아이콘이 동시에 존재할 수 없음)

3. 윈도우
  - 껏다가 켯다가 할 수 있어야 됨.

4. 기타
  - 디자인
